

Welcome to retico-wozmic's documentation!
=======================================

Retico is a framework to build incremental and modular dialog systems.
Retico-wozmic provide any Retico-based system with a Wizard-Of-Oz (WOZ) Microphone Module, that sends
audio IUS corresponding to an audio file (`audios/hello16k.wav`) each time the `M` key is pressed.
This Module can be useful to debug many dialog system behvaiors.
